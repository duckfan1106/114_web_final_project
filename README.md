# Social Media Content Sharing Platform

A full-stack web application for sharing content, built with MERN stack (MongoDB, Express, React/Next.js, Node.js).

## Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB (Local or Atlas)

### Installation

1. **Clone the repo**
   ```bash
   git clone <repo-url>
   cd project-name
   ```

2. **Setup Backend**
   ```bash
   cd backend
   npm install
   # Create .env file if needed, defaults to local mongodb
   npm run dev
   ```

3. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
